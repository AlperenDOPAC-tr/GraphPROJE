# r3f-vite-starter
A boilerplate to build R3F projects

## Gereksinimler
- Node.js (en az 18.x önerilir)
- npm veya yarn

## İlk kez çalıştırma
1. Bu proje klasörüne gidin:
   ```bash
   ```
2. Bağımlılıkları yükleyin:
   - `npm` kullanıyorsanız:
     ```bash
     npm install
     ```
   - `yarn` kullanıyorsanız:
     ```bash
     yarn install
     ```
3. Geliştirme sunucusunu başlatın:
   - `npm` için:
     ```bash
     npm run dev
     ```
   - `yarn` için:
     ```bash
     yarn dev
     ```
4. Tarayıcıda açın:
   - `http://localhost:5173`

## Bu proje hangi kütüphaneleri kullanıyor
- `react`
- `react-dom`
- `three`
- `@react-three/fiber`
- `@react-three/drei`
- `@react-three/cannon`
- `@react-three/rapier`
- `vite`
- `@vitejs/plugin-react`

## Faydalı komutlar
- Üretim build'i oluşturmak için:
  ```bash
  npm run build
  ```
  veya
  ```bash
  yarn build
  ```
- Build sonrası önizleme için:
  ```bash
  npm run preview
  ```
  veya
  ```bash
  yarn preview
  ```

> Not: Bilgisayarında çalıştırmak için Node.js kurulumu yapılmış olmalı. npm veya yarn kuruluysa proje hemen açılır.
